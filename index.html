Install From Source

Clone the repo and build DALIOS natively on your platform.

WINDOWS (PowerShell)

git clone https://github.com/defthrets/dalio-trading-system.git
cd dalio-trading-system
pip install -r requirements.txt
.\build.bat

LINUX (Terminal)

git clone https://github.com/defthrets/dalio-trading-system.git
cd dalio-trading-system
pip install -r requirements.txt
chmod +x build.sh && ./build.sh

macOS (Terminal)

git clone https://github.com/defthrets/dalio-trading-system.git
cd dalio-trading-system
pip install -r requirements.txt
chmod +x build-mac.sh && ./build-mac.sh
// Core Principles

Dalio's Economic Machine

Every decision is grounded in Ray Dalio's proven macro-economic framework.

▲ Rising Growth + Rising Inflation

Overweight commodities, TIPS, emerging market equities. The economy is heating up — ride momentum while hedging inflation risk.

▲ Rising Growth + Falling Inflation

Overweight equities, corporate bonds. The goldilocks environment — lean into risk assets as conditions favor broad expansion.

▼ Falling Growth + Rising Inflation

Overweight gold, commodities, inflation-linked bonds. Stagflation territory — protect capital and hedge purchasing power erosion.

▼ Falling Growth + Falling Inflation

Overweight nominal bonds, defensive equities. Deflationary risk — prioritize capital preservation and safe-haven positions.
